【パッケージ内容】
本パッケージ内のファイル構成は以下の通りです。
スタイルファイルの使い方については使用例を参照してください。

《iscie-style-sjis》
  │
  ├ readme.txt   このファイル
  │
  ├《style》     スタイルファイル
  │  │
  │  └ *.cls, *.sty, *.fd （改変せずに使用してください）
  │
  └《template》  テンプレート集
      │
      ├《paper》      論文誌投稿用
      │  │
      │  ├ figure-p.pdf   作図見本
      │  ├ paper.tex      日本語論文（兼 使用例）
      │  ├ paper.pdf      上記のPDF版
      │  ├ paper_e.tex    英語論文（兼 使用例）
      │  ├ paper_e.pdf    上記のPDF版
      │  ├ tmp.eps        （使用例で使用するファイル）
      │  ├ tmp.tex        （使用例で使用するファイル）
      │  └ tmp_e.tex      （使用例で使用するファイル）
      │
      └《journal》    会誌寄稿用
          │
          ├ figure-j.pdf   作図見本
          ├ exposit.tex    解説, 総説, 展望（兼 使用例）
          ├ exposit.pdf    上記のPDF版
          ├ bexposit.tex   ミニ解説
          ├ book.tex       書評
          ├ comments.tex   読者の広場
          ├ essay.tex      編集後記, 前記
          ├ intervie.tex   対談, 討論, 電子メール討論, 座談会
          ├ lecture.tex    講座
          ├ report.tex     国際会議の報告
          ├ rgroup.tex     研究活動紹介
          ├ sciforum.tex   SCIフォーラム
          ├ mondou.tex     アイ・サイ問答教室
          ├ tmp.eps        （使用例で使用するファイル）
          ├ tmp.tex        （使用例で使用するファイル）
          └ tmp-short.tex  （使用例で使用するファイル）


【著者略歴について】
下記の内容（年次は西暦表示）を250字以内にまとめ、氏名，所属，会員種別とともに記載してください。
写真を掲載する場合は、顔写真（4cm x 3cm以上）を使用してください。

  1. 卒業年月，最終卒業校
  2. 勤務経歴
  3. 担当業務内容，研究内容
  4. 学位，受賞事項
  5. 本会以外の所属学会名
